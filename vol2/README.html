一

会议室里灯火通明，桌子上的黑色电话机那延迟严重的电音听起来有些渗人。

「 See … you … then … 」

「 西柚 」

门铃关掉电话，长长的叹了一口气。门铃其实是个女生，姓东门，名灵。于是大家都叫她门铃。

「 搞什么突然袭击，互联网分析师了不起啊，微博上10个SB里就有8个分析师。而且明明是中国人说个毛英语啊，又不是投行的。 」许先合起他那贴满天津风大腿的MacBook，把憋了一肚子的话抖出来。

今天的会开了快6个小时，他没说上一句话，作为公司的核心技术，他在场只是以备对方问技术问题。

「 嘛，遇上这种事，谁都会觉得不对劲吧，问问也是他们份内的事。 」门铃说得有些黯然。这家原本颇有名气的移动互联网公司两周前突然被一个餐饮集团全资收购了。

「 我下周打算辞职了，我对开发订餐系统不感兴趣。你那边的需求可能要堆积一段时间了。 」许先说完有些幸灾乐祸。

「 老大没告诉你？今天是我在公司最后一天，我对大堂经理也不感兴趣。 」门铃摊开手。

两人相视一笑，这大概是他们这半年来做得最合拍的事情了。

每个公司都有程序员和产品经理的战争，而他们正是各自的大将。不过现在，战场没了。

走出会议室，公司已经空无一人，门铃看了眼手机，凌晨3点。

二

「 不许往上看哦！ 」门铃略显羞涩的说到。

「 好的。 」许先抬起头看着正踩在他肩上的合法萝莉，心中一阵悸动。

「 不行不行，我怎么能喜欢上三次元的产品经理呢？就算是蓝白条纹的产品经理也不行。这只是吊桥效应。」

在从会议室回工位的途中，他们被困在了电梯中。所幸的是电梯门已经被锋利的Air撬开，只要翻过近2米的隔层就可以顺利脱险了。

就这么件事许先足足花了10多分钟，他太缺乏锻炼了，在地板上滚了两圈后，终于站了起来。

「 这层是公司总裁办公室，平时电梯是不停的。」 许先一边领着门铃走向楼梯，一边给她科普。这里已经好久没用了，只有资格很老的员工才来过。

「 那边有光哎，我们去借个手电吧！ 」门铃卡啦一声推开了那扇门，然后愣住了。

办公桌上只开着台灯，穿着黑色晚礼服的女人坐在西服男怀里，脚盘在他身后，前后晃动着，发出断断续续的呻吟。

面对突如其来的门铃，他们并未分心，西服男一手搂住晚礼服的腰，另一只手端起酒杯喝了一口，气喘吁吁的感叹到，「 你这个磨人的小妖精… 」

许先一个箭步上去，冲着屋里大声说到：「 老黄，说过多少次别TMD在公司看爱情动作片，再有下次让你走人啊。电梯坏了，打个电话叫人修！ 」

然后拽着门铃往楼梯跑过去。

「 刚才那人不是新来的董事长吗，为什么你叫他老黄？ 」跑得满脸潮红的门铃上气不接下气的问到。

三

「 他叫你老黄呢！ 」晚礼服咬着西服男的耳朵说。

「 你知道我最喜欢两种人，一种是够性感的女人，一种是够聪明的男人。 」西服男把晚礼服放到沙发上，理了下领带，然后拿起桌上的对讲机：「 带过来。 」

门铃和许先被两个真枪实弹的保安押着，带了进来。

「 小许，小灵，才一天没见你们就不认识我了啊？ 」

「 刚才光线暗，没看清，更没想到董事长您半夜还在为公司操劳，真是抱歉。 」门铃终于恢复了灵牙利齿。

「 既然看见了，那就没办法了。 」董事长抿了一口酒，拿起桌上的左轮手枪。

「 我们可不是什么餐饮集团，我们是专门从事高智商犯罪的比特帮。 」带保安进来的眼镜推了推自己的鼻梁，冷冷的说到。

「 我们不是叫菜刀帮吗？ 」一个保安悄悄的问旁边的帮兄。

「 都和你说了别把年假攒一块休，上个月就改了名字了啊。 」另一个保安狠狠踹了他一脚。

「 咔咔咔咔咔咔 」董事长把左轮的转盘撸了一圈，然后走过来。

「 这里边有一颗子弹，你们两人里边必须有一个人打对方一枪，谁来？ 」

「 我！ 」

「 她！ 」

「 哦？我还是第一次遇到决定得这么干脆的。 」董事长把抢放到门铃面前。

门铃面露愧色，许先笑笑，安慰她说，「 没事的，反正我们开发一直都是帮你们产品背黑锅挡枪子儿的，也不在乎多一次了。 」

拿起枪，门铃的手抖得厉害，她放下又拿起，拿起又放下，足足折腾了5分钟。

晚礼服饶有兴致的研究门铃和许先的表情，可董事长有些不耐烦了，从保安那儿拿过一挺冲锋枪对着门铃。「 我数十下，如果你没开枪，我就开枪。 」

「 十，九 … 」

「 你是我的小丫小苹果呀 ♫ 怎么爱你都不呀不嫌多呀 ♫ 」

董事长满脸无奈，把枪扔还给保安，坐回桌前。

眼镜拿出土豪金听了下，对董事长说，「 老大，青姐要找的人带过来了。 」

「 说了多少遍，别叫我老大，叫我总裁！ 」

「 好的老大，这两个人要怎么办？ 」

「 先关到壁柜里边吧。 」

「 好的老大。 」

「 等等，先把那女的的袜子脱下来。 」

「 老大你不是只喜欢黑丝的吗？ 」

「 那是用来堵他们嘴的，你们这帮废物。还有，叫我总裁。 」

「 好的老大。 」

四

「 你，你… 我… 」

董事长看了晚礼服，也就是小青，或者叫Ann，一眼。Ann耸了耸肩。

「 好吧，那现在让我们用中文来聊聊这些钱怎么花的。 」

「 员工工资我一直拖着没发。 」

「 Ann总，创业公司不容易啊，能省的我都省了。」莫庭抹去头上的汗滴，继续说到。

「 匿名社交是个新东西，得强推。为了能上应用榜前十，我们找了家专业刷榜公司，每天5万，两个月下来花了300万。他们也给那些大公司刷，也是这个价，没坑我们。 」

「 虽然用户留存比较差，但每天都有大量的新安装，我们也赚足了眼球。本来下一轮投资Term sheet都谈好3家了，结果国外抄袭我们那家公司反过来把我们给告了，TMD，苹果也太不讲理了，把我们下架了。这么一来，三家投资方都闪人了。 」

「 不是都签了Term sheet吗？怎么能反悔！我最讨厌这些没有契约精神的中国人了！ 」董事长有点生气了。

「 他们是外国人… 」Ann耸耸肩。

「 情况我大概了解了，你也算是尽心了。 」董事长点了点头，「 但是Ann投你的钱是从我客户那里拿的，这笔钱下个月必须要还出来。 」

「 老大，这和投资协议上写的不同啊！ 」

「 你们还有投资协议？不是Ann让你睡了，你就让她投了么？ 」董事长向Ann望过去。

「 最近投了太多家公司，我忘了有没有和他上床了。 」Ann说得轻描淡写。

「 总之，一个月内，500万务必还回来。 」董事长摆弄着桌上的子弹，一个字一个字的说。

「 你这是逼我抢银行啊！ 」莫庭快哭了。

董事长摸着脑袋，大概是回忆起了自己的青葱时代，自顾自的笑了。

「 就这么定了。我支持你去抢银行，抢来的钱我只要500万，多的给你，毕竟你只欠我们这么多，契约精神我还是有的。 」

一打响指，眼镜把门铃和许先从壁柜里边拉了出来。

「 这两个人给你做帮手。 」

「 你不是说支持我们吗？那借我们50万活动资金，到时候一起还给你。另外我需要去香港查点资料，内地很多站上不去，你知道的。 」许先已经对需求来之不拒了，只要听到需求，他就会条件反射的去设计解决方案。他早就不再去判断需求是否合理了，这种事从来轮不到技术说话。

「 我就喜欢聪明人。 」董事长右脸笑了笑，「 钱没问题，不过查资料goagent不就行了么？ 」

「 上周防火墙升级了，很多站被封锁得很厉害，goagent已经废了。 」

「 狗日的方XX，这么做坏事不得好死的。老子学了几个月才学会用goagent。 」董事长仰望着窗外的星空，似乎想起了一段痛苦的回忆。

五分钟后，董事长转过身。

所有人都瞪大了眼睛。

「 搞毛啊，给他们拍裸照！给几十万让到处跑，总得留点把柄在我们手上吧？ 」

「 那女的就不用脱了，胸比我还小，脱了也没什么看的，让她握着两个男的下边就够了。 」董事长显然是巨乳派。

「 不如让那个程序员握着莫总监吧？ 」Ann的兴趣。

她说，为了追寻自己想要的生活，她背井离乡漂在北京，但她过得最快乐的日子，偏偏却还是被别人安排的。这大概就是宿命吧。

八

郁小梦对着镜子又抹了抹唇膏，依然是甜橙味的。用手最后理了下前额的刘海，在玻璃前坐直了。

下午4点，银行里边没有一个顾客。小梦看着玻璃上那张还能漂亮两年的脸，开始发起呆来。

因为家里的关系，高中毕业后她就毫无悬念的进了这家安稳的银行。

所谓安稳，就是只要每天说一样的话，每天做一样的事，每月就可以领到不错的薪水。

所谓安稳，就是气势汹汹的女主管已经变成称霸广场的大妈，帅气的男同事变成了秃顶的男同志，2楼的保安都已经换了十几茬，1楼的员工不要说换人，就连办公的位置都没换过。

所谓安稳，就是没有惊喜，几年的时间回忆起来就像一天。唯一不同的，是那些取钱存钱的客户，老实说，小梦从没记住过他们的脸。

但接下来这个客户却注定要让小梦印象深刻。

他是位黄头发，蓝眼睛的外国人。7分短裤，沙滩拖鞋，提着两个红白交织的LV包包。

「 窝要蠢钱！ 」

LV放上柜台一拉开，满满一袋零散的百元人民币。

七分裤把另外一个包放到座位下，等着小梦清理钱数。包里边的钱多而乱，即时使用点钞机，也花了快40分钟。

排在后边的民工等不及了，问到 「 还有好久哦？你还要存好多嘛？ 」

「 只蠢这一包。 」

「 存好了，这是您的银行卡和存折，请收好。先生您还要办理其他业务吗？ 」

「 不用谢，啊，我的Phone响了。 」

七分裤把手机用脑袋夹住，一手拿着银行卡，一手提着LV哇哇哇的走了出去。

后边的民工取完钱，发现自己的编织袋变小了点，拉开一看，顿时吓得坐到了地上。

「 囊个回事捏？老子一编织袋的榨菜囊个都变成钱哒？ 」

大堂的保安过来一看：

「 啥子编织袋嘛，咧个是LV的包包塞。 」

本来就要关门的银行一下子热闹了起来，柜台经理一看要坏事，赶紧叫保安。

「 赶快去把那个老外追回来。 」

「 晓不得往那边跑的。 」

「 那你也去追，他提的你的包包，你认得出来些。 」经理一急，开始支使民工干活了。

「 那咧些钱囊个办奈？ 」民工看着手上的包包。

「 我们先锁到柜台里头，赶快去。 」经理夺过包，穿过几道铁门，把它放在办公桌边。

一个小时后，民工和老外都没有回来，经理已经接到三个电话催她打麻将了。

「 打个电话和吴哥请示哈嘛？ 」小梦建议到，吴哥是这家银行的总经理，但在家里，柜台经理才是他领导。

结果当然是柜台经理愉快的去打麻将了。小梦吃完路边摊，慢悠悠的往家走。

「 说起来，那个老外怎么连1米7都没有呢，日本人咩？ 」她突然自言自语了这么一句。

锁在银行的LV编织袋突然翻了个身，Polly从里边爬了出来。

「 信号有点弱啊。 」它抬起前爪揉了揉脸。

九

「 我也觉得你这外国人矮了点。 」门铃看着正在卸妆的许先，给莫庭递过去一杯水，「 倒是莫总演得很地道，连方言都说得很棒啊。 」

「 化妆太狠了，直接把我扔民工旅馆住了两天，还偷了这一身衣服。 」莫庭接过水杯，咕嘟咕嘟一大口。

「 没想到你们比特帮居然有这么懂化妆的人啊。 」许先也很感叹。

「 不是我们的人，那人就是拍电视剧的化妆师。演艺圈的嘛，经常到我们那里买粉，一来二往就熟识了。 」眼镜说得很淡定。

「 哇喔，那你们是不是还认识很多明星？ 」门铃又摆出了星星眼表情。

「 别闹，赶紧收拾下，下半场要开始了。 」Polly不在，条形码只好在iPad上板着脸，「 已经连上银行内网，接下来交给你了，许Sir。 」

许先把Air抱到酒店阳台，播放器的音量开到最大，然后罩上那印着两个b的耳机：「 2个小时别和我说话。 」

门铃看着许先那被屏幕反光时不时点亮的眼睛，那个清秀的书生已然变得陌生。

杀气。

这个只在武侠小说里读过的词变得具体起来。

我要从这里杀出一条血路，挡我者死。

佛来斩佛，魔来斩魔。

门铃突然明白了这个二次元的宅男为什么能成为公司的技术领袖——狼性，他身上有着怎么加班也加不出来的狼性。

「 Smart is the new sexy , huh? 」条形码对着发呆的门铃说。

「 长得丑的男人都这么安慰自己，毁过容的也是。 」门铃对着条形码吐吐舌头。

十

只用了半个小时，许先就回来了。

「 这么快？几年不见功力大涨啊。 」条形码不得不佩服。

「 一家银行居然用智能路由，还装了个迅雷离线在上边，一堆GV种子。 」许先摇着头说，「 拿完Root刷了个Web Shell，用起来就跟自家后台似的。 」

「 你想要，种子？ 」门铃发现眼镜欲言又止。

「 不要了，种子没用，离线早就废了，可惜了我刚续完一年的会员。 」眼镜已经看破红尘。

「 安保系统上周官方升级了一个版本，我对比了下，找到了一个高权限漏洞。现在整个系统已经被我们控制了，但是我们要想办法干掉那个4G报警器。 」

「 4G很高科技所以难搞么？ 」门铃明明对所有新科技都很好奇，却完全没有常识，这大概是产品经理的职业病。

「 其实就和我的4G手机一样，只是走了独立的信道，所以黑掉路由对它没影响。 」莫庭解答到。

「 这设备做振动监测的，不太容易触发，可以先不管它。 」

「 我擦，振动监测？你丫不早说，我刚让Polly把它从机箱顶上踹下来…… 」条形码一急，港台腔都没了。

「 收队，回北京。 」莫庭拿出编织袋开始收拾东西。

「 就这样？ 」门铃不甘心，为了这事他们足足准备了6天。

「 再不走就走不了了，警察很快就会包围银行，然后开始排查可疑人员。 」许先强调必要性。

「 以前我们都真刀真枪的进入，出这种事命都没了。现在这样已经算是很小的损失了。这个我要记下来，在帮里推广。 」眼镜从收拾好的包里拿出笔和纸，开始划拉起来。

「 你们赶紧，我去退房。 」莫庭已经收拾完。

一行人站在酒店门口，莫庭打电话叫车，却死活打不通。

「 你也是4G迈？ 」酒店保安凑了过来，「 小区的大妈因为媳妇怀孕，找了一群棒棒把信号塔拆了，从昨天下午开始就一直没信号哒。 」

许先眼睛一亮，给条形码发了条短信，他没用4G，因为手机不支持。

「 的确没信号。 」很快收到了Polly的确认。

十一

一辆解放悄然停在了银行后门。Polly吐出的乙醚很快迷倒了二楼的保安们，许先一敲键盘，咔嚓，整个金库门户洞开。

几个黑衣人鱼贯而入，将半个集装箱高的人民币拖回车后的货架上，整个过程只用了10分钟不到。

放下电脑，许先和其他几个人一起开始把一匝匝的钱包上事先印好的「书」皮。

「 要想不被人发现，就要藏木于林。 」这是门铃的点子。虽然谁都不嫌钱多，但是搬动上亿的现金却是很麻烦的。

红红的人民币用特制的书皮一包，立马变成一块板砖，周围用货真价实的红砖一围，在这夜色下还真难辨真伪。

砖车很快离开城区上了国道，消失在夜色中。

再也没有人见过这辆车。

十二

郁小梦涨薪了。

银行被盗后，县委书记过来视察，决定不往上呈报，县里内部处理了。二层本来就是县里小金库，和银行没啥关系。

最近新闻联播里隔三差五有官员落马，2个亿的篓子可不是闹着玩的。

「 留得青山在，不怕没财烧。 」书记是这么安慰县委班子的。银行的员工薪水都翻了一倍，条件是对这件事情绝对保密。

「 保密？这事说出去也没人信的。」午饭的时候小梦和经理谈到这事。

「 几年敛出来的2个亿丢了，书记脸色都没变一下，这种人以后肯定能成大事的。 」经理一脸崇拜，「 我看到了我们国家的希望啊。 」

小梦突然觉得当个小角色也挺好的，她只有一颗玻璃般的小心脏。

十三

一个月后。

庆功宴在公海的游轮上举行，夕阳西落，把船上那些人脸和高脚酒杯都镀上了金色。

还给董事长550万后，许先和莫庭成立了一家比特币支付公司，神不知鬼不觉的把分来的钱洗得很干净。

莫庭半开玩笑的说很诧异自己还能活到现在，他以为事成后一定会被杀人灭口。

「 你们太小瞧我了。 」董事长和莫庭碰完杯，指着一望无际的大海说到，「 性格决定命运，气度影响格局。要做大事，点到为止，不能贪得无厌。」

「 一听就是你那帮中央政治学院的老同学们教的吧？ 」Ann 今天依然穿着露出整个后背的黑色晚礼服。

「 多点朋友总没有错嘛，尤其是彼此都有对方底牌的，对吧小许？ 」董事长笑意盈盈。

「 董事长说得太对了。 」挽着许先胳膊的门铃咯咯的笑到。以前她的爱好是改变世界，现在她的爱好是每天早上看余额宝的收益，然后开始买包包。

赌场开局，游客们纷纷回到舱内。许先和门铃站在船舷边，小西服、百褶裙，阳光在甲板上切出来一个完美的剪影。

「 你觉得我今天漂亮吗？ 」门铃举着酒杯问许先。

「 我觉得 」许先一扭头，「 有点不对劲。 」

一群鱼从天上掉下来砸到甲板上，只一瞬，狂风夹着巨浪呼啸而至。

几千吨的游轮像个玩具，被一下子抛到了几百米的高空。

许先一把把门铃拽进怀里，死死的搂住她。

台风来了，猪都能飞上天，但是台风过去了，猪会怎么样呢。

想到这里，许先突然笑了，他捧住门铃那惊慌的脸，冲着那樱桃红唇，优雅的吻了下去。

台风过后的事情，让猪去操心吧。

十四

书记步履蹒跚地把盘子放到桌子上，然后小心的把脚上的铁链顺进去，抬头看着监狱里唯一的娱乐。

老旧的电视上正播放着新闻：

「 八号台风浣熊昨日从我国海岸线掠过，后转向日本，未造成人员伤亡。但公海一游轮误入台风口，无人生还。」
